# Support

scankit is a small, focused Go library maintained on a best-effort basis.

## Getting help

- **Usage questions** — read the [README](README.md) and the per-package guides under
  [`docs/`](docs/), or the API on
  [pkg.go.dev](https://pkg.go.dev/github.com/stephrobert/scankit).
- **Bugs** — open a [bug report](https://github.com/stephrobert/scankit/issues/new/choose).
- **Feature ideas** — open a feature request; note that scankit intentionally stays small,
  so product-specific behaviour belongs in the consuming tools (pepin, pitstop, pavois).
- **Security vulnerabilities** — do **not** open a public issue; follow
  [SECURITY.md](SECURITY.md).

## Response expectations

This is a volunteer-maintained project. Issues and pull requests are reviewed as time
allows; there is no guaranteed response time except for security reports, which are
acknowledged within 5 business days (see [SECURITY.md](SECURITY.md)).
